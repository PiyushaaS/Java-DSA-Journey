# Sorting Algorithms (Java)

This folder contains implementations of common **sorting algorithms** written in Java as part of my Data Structures and Algorithms (DSA) practice.

Algorithms included:

* Bubble Sort
* Selection Sort
* Merge Sort
* Quick Sort

These programs help in understanding different sorting techniques, their logic, and how elements are arranged in ascending order.

More sorting algorithms will be added as I continue learning and practicing DSA.
