# Recursion

This folder contains Java programs that demonstrate recursion concepts.

Recursion is a technique where a function calls itself to solve a smaller instance of a problem.

## Problems Implemented

- Print n to 1 using backtracking
- Factorial using recursion
- Sum of n numbers using functional recursion
- Print 1 to N using backtracking
- Print numbers from 1 to N using recursion
- Sum of first N numbers
- Print numbers from N to 1 using recursion
- Reverse an array using recursion
- To check if the string is palindrome using recursion
- Fibonacci using recursion
  
## Language Used
Java
