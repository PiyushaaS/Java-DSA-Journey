# Hashing in Java – DSA Practice

This folder contains my practice problems on **Hashing** while learning **Data Structures and Algorithms in Java**.

Hashing is a powerful technique used to store and retrieve data efficiently using structures like **HashMap** and **HashSet**. Many problems involving **frequency counting, duplicates, and fast lookups** can be solved using hashing.

## Concepts Practiced

* Frequency counting
* Counting occurrences of elements
* Checking conditions using stored frequencies
* Digit count

## Problems Included

| File Name                     | Description                                           |
| ----------------------------- | ----------------------------------------------------- |
| `CountFrequency.java`         | Counts the frequency of elements using hashing        |
| `HighestOccuringElement.java` | Finds the element with the highest frequency          |
| `PrimeFrequency.java`         | Checks if any element appears a prime number of times |
| `DigitCount-Leetcode.java`    | LeetCode problem solved using hashing                 |

## Technologies Used

* Java
* HashMap
* HashSet
* Basic DSA concepts

## Goal

The goal of this folder is to **build strong problem-solving skills using hashing** and understand how hash-based data structures improve time complexity.
